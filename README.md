## pigeon

> pigeon 是一款基于七牛云的文档管理工具，目的是将你编写的文档资料上传至对应的云服务空间中

![Gbos2j.png](https://s1.ax1x.com/2020/04/11/Gbos2j.png)



![GbTpzd.png](https://s1.ax1x.com/2020/04/11/GbTpzd.png)

### 打包

```java
// 尽量翻墙，保证资源下载成功
npm run dist
```

云服务商支持

- 七牛云
- 待补充
